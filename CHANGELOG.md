# Changelog

Historial de versiones del Sistema de Adopcion de Mascotas.

## [1.0.0] - 2025-09-26
### Added
- Sistema completo de adopcion de mascotas funcional
- Interfaz de usuario completa HTML/CSS/JavaScript
- Backend en PHP con funcionalidades CRUD
- Base de datos MySQL con tablas de usuarios mascotas y adopciones
- Panel administrativo para gestion
- Sistema de registro y autenticacion de usuarios
- Catalogo de mascotas disponibles para adopcion
- Proceso de solicitud de adopcion
- Configuracion XAMPP para entorno de desarrollo
- Estructura de carpetas organizada del proyecto
