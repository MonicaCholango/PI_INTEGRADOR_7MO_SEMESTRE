# PLINTEGRADOR_7MO_SEMESTRE

Sistema de Adopcion de Mascotas - Proyecto Integrador Septimo Semestre

## Descripcion
Sistema web completo para gestion de adopciones de mascotas.

## Estructura del Proyecto
- Aplicacion/AppAdopciones/ - Aplicacion web principal
- Base de datos/ - Scripts SQL y configuracion de base de datos
- Xamp/Pae/ - Configuracion del entorno XAMPP
- docs/ - Documentacion del proyecto

## Estado Actual
- Sistema funcional completo
- Base de datos implementada
- Interfaz de usuario terminada

## Tecnologias Utilizadas
- Frontend: HTML5 CSS3 JavaScript
- Backend: PHP
- Base de datos: MySQL
- Servidor: XAMPP
