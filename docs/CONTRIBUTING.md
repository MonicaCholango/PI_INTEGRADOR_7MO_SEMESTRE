# Guia de Contribucion

## Como Contribuir al Proyecto

### 1. Configuracion del Entorno
1. Fork el repositorio
2. Clona tu fork
3. Instala XAMPP
4. Configura tu entorno de desarrollo

### 2. Flujo de Trabajo
1. Crea un issue describiendo el cambio
2. Crea una rama desde main
3. Realiza tus cambios
4. Haz commits descriptivos
5. Abre un Pull Request

### 3. Convenciones
- Ramas: feat/ fix/ docs/ chore/
- Commits: feat: descripcion #issue

### 4. Testing
- Prueba tus cambios localmente
- Verifica que la aplicacion funciona en XAMPP
