# Plan de Gestión de Configuración (CMP)
## Sistema de Adopción de Mascotas PAE

### 1. Información del Proyecto

**Nombre:** Sistema de Adopción de Mascotas PAE  
**Versión:** 1.1.0  
**Periodo:** Julio - Septiembre 2025  
**Equipo:**
- Monica Cholango Collaguazo - Líder de Proyecto, Backend y Frontend
- Byron Torres - Desarrollador Backend 
- Jose Arevalo - Desarrollador Frontend

### 2. Arquitectura del Sistema

**Stack Tecnológico:**
- Frontend: Angular + Ionic Framework
- Backend: PHP con API RESTful
- Base de Datos: MySQL (15 tablas)
- Servidor: XAMPP (Apache + MySQL + PHP)
- Control de Versiones: Git + GitHub

**Componentes Principales:**
- Aplicación Web Responsive
- API con 40+ endpoints
- Base de datos relacional
- Sistema de autenticación
- Módulo de adopciones
- Panel administrativo
- Sistema de eventos
- Módulo de denuncias

### 3. Funcionalidades Implementadas

**Para Adoptantes:**
- Registro y autenticación
- Búsqueda de mascotas disponibles
- Solicitud de adopción
- Seguimiento de solicitudes
- Reporte de denuncias
- Visualización de eventos

**Para Administradores:**
- Dashboard con estadísticas
- Gestión de mascotas (CRUD)
- Procesamiento de solicitudes
- Gestión de usuarios
- Creación de eventos
- Generación de reportes

### 4. Proceso de Desarrollo

**Metodología:**
- Desarrollo incremental
- Control de versiones con Git
- Revisión de código en Pull Requests
- Testing manual y automatizado
- Documentación continua

**Herramientas:**
- GitHub para repositorio y issues
- XAMPP para desarrollo local
- phpMyAdmin para gestión de BD
- Visual Studio Code como IDE
